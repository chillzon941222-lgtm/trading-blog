足跡圖交易：辨識被套部位，站在對的那一側

被套的人遲早要逃跑。足跡圖讓你看見他們被困在哪裡，然後等著他們推動市場。

chillzon2025

什麼是「被套交易」

先想一個生活場景：你搭電梯，進去之後才發現搭錯方向了。電梯往上，你卻要去地下室。你能做的只有一件事——等電梯到頂樓，再趕快跑出去換另一部。

市場裡的「被套者」也一樣。他們在錯誤的方向開了倉，現在方向和他們預期的相反，帳面虧損越來越大。遲早，他們必須認輸——平倉、認賠、跑出去。

而他們平倉的行為，會產生市價單，推動價格繼續朝對他們不利的方向移動。

足跡圖讓你看見這些被套者在哪裡——他們的進場價在哪裡，他們的止損聚集在哪裡。

足跡圖複習：Bid × Ask

每個價格節點顯示兩個數字：

數字代表意思
Bid（左）市價賣出量積極賣方，打到限價買單
Ask（右）市價買入量積極買方，打到限價賣單
DeltaAsk - Bid正值=買方更積極，負值=賣方更積極

被套多頭（Trapped Longs）

某群人在高點看好行情，買進了。結果市場往下走，他們被套了。

足跡圖上，你會在某個高點看到大量 Ask 量（市價買入）——那就是他們進場的地方。

當價格再次上漲到那個位置，他們會鬆一口氣準備解套平倉（市價賣出）。這些解套的賣單，會成為上漲的阻力，甚至讓市場在那裡再次反轉向下。

被套多頭的止損是市價賣出單，一旦觸發就成為推動下跌的燃料

被套空頭（Trapped Shorts）

相反的情況：某群人在低點做空，結果市場往上走，他們被套了。他們的止損是市價買入單——一旦觸發，就成為推動上漲的燃料。

足跡圖上，你會在某個低點看到大量 Bid 量（市價賣出）——那是他們做空進場的地方。當價格再次跌回那裡，注意他們是否開始認輸（市價買入回補）。

三種關鍵的足跡圖型態

型態一：吸收（Absorption）

大量的市價單湧入（買或賣），但價格沒有移動。代表有大型限價單在默默承接——這是主力在悄悄建倉。

下跌中的吸收（底部訊號）：
  某個價格節點：Bid量爆增，但價格不跌
  → 大量賣單被限價買單吸收
  → 底部，考慮做多

型態二：衰竭（Exhaustion）

推動價格的力道越來越小。每根 K 棒的 Delta 在縮小，最高/低點的成交量在萎縮。就像一個人衝刺，到終點前越跑越慢。

上漲衰竭（頂部訊號）：
  連續幾根K棒的 Ask 量：3500 → 2200 → 800
  最高點的 Bid 和 Ask 都是微小量（微單）
  → 買方動能耗盡，拍賣終止

型態三：不平衡（Imbalance）

足跡圖上某個節點，其中一側的量遠大於另一側（通常超過 3:1），而且成交量很大。這代表市場對這個價格有強烈的方向性共識。

吸收 = 有大量對手盤，主力建倉｜衰竭 = 推力耗盡，即將反轉｜不平衡 = 方向明確，趨勢確認

Order Flow Delta：足跡圖的靈魂數字

在深入 CVD 之前，你必須先搞清楚一個核心概念——Delta（Δ）差值。

足跡圖（Footprint Chart）在每個價格節點上，都會顯示兩個數字：

左側數字：Bid 成交量（賣方主動，使用市價賣單擊穿 Bid 掛單）
右側數字：Ask 成交量（買方主動，使用市價買單擊穿 Ask 掛單）

Delta = Ask 成交量 − Bid 成交量

舉個生活化的例子：假設某個價格節點上，Ask 成交了 350 口、Bid 成交了 200 口，那麼：

Delta = 350 − 200 = +150（正 Delta，買方更積極）

反過來，如果 Bid 成交了 400 口、Ask 成交了 150 口：

Delta = 150 − 400 = −250（負 Delta，賣方更積極）
Delta 值意義市場含意
正 Delta（+）Ask 成交 > Bid 成交主動買方較多，積極追買
負 Delta（−）Bid 成交 > Ask 成交主動賣方較多，積極砸盤
Delta 接近 0買賣勢均力敵平衡狀態，方向不明

為什麼 Delta 差值很重要？因為傳統的 K 棒只告訴你「開高低收」，完全看不出誰更積極。Delta 讓你看到的是方向性的主動力道——是買方在積極追市，還是賣方在主動砸盤。

關鍵認知：一根上漲的 K 棒，不代表買方更積極。如果這根 K 棒的 Delta 是負值，代表雖然價格上漲了，但其實是賣方在砸盤，卻有更大的掛單在接住賣壓——這反而是強勢吸收，是更看漲的訊號（吸收型態）。

有了 Delta 的概念之後，接下來就可以理解 CVD 了——它只是把 Delta 累計相加，形成一條可以觀察趨勢的線。

CVD：累計成交量差

CVD（Cumulative Volume Delta，累計成交量差）是把每根 K 棒的 Delta（Ask 成交量 - Bid 成交量）累計加總，畫成一條線。

CVD 讓你看到的是：從某個時間點以來，買方主動力量的累積是否大於賣方。

CVD 上升：累計下來，主動買方比賣方多 → 買方佔優
CVD 下降：累計下來，主動賣方比買方多 → 賣方佔優

Delta 背離（Delta Divergence）是最關鍵的訊號：
價格創新高 + CVD 卻沒有跟著創新高 → 背離，上漲缺乏主動買方支撐，潛在反轉
價格創新低 + CVD 卻沒有跟著創新低 → 背離，下跌缺乏主動賣方支撐，潛在反彈

比喻：CVD 就像一場拔河的計分板。就算繩子目前在偏多方，但如果最近幾回合多方的力道在衰退，之後可能被拉回來。

Stacked Imbalance：堆疊的方向性訊號

單一的 Imbalance（不平衡）已經是強訊號。當連續多個價格節點都出現同一方向的 Imbalance，叫做 Stacked Imbalance（堆疊不平衡）。

這代表：市場對這個方向有壓倒性的一致共識，不只是一個點在發力，而是整個區間都在朝同一個方向推進。

Stacked Imbalance 的使用：

  • 形成時：順勢進場，這是動能最強的區域
  • 被回測時：通常是強力的支撐/阻力，等待失敗訊號或突破確認
  • 被穿越時：可能是趨勢反轉的訊號，需要重新評估方向

完整交易 SOP

盤前準備

1. 標記前日 POC / VAH / VAL
2. 標記所有 Naked POC（近5-10日）
3. 記錄前日 VWAP 位置
4. 確認今日開盤在 Value Area 內還是外

盤中執行

開盤15分鐘：觀察，不急著下單

看 VWAP 上下：決定今天偏多還是偏空

等待價格到達關鍵節點：
  → 到達節點後，看足跡圖型態
  → 吸收：等吸收完成後的反向K棒
  → 衰竭：最高/低點微單出現，確認後入場
  → 不平衡：順勢，目標下一個 HVN

止損設定：節點另一側 2-5 ticks
目標：下一個 HVN 或 POC

出場規則

到達目標 HVN：先出50%，剩餘看能否突破
出現反向吸收或衰竭：全部出場
接近 VWAP 或 POC：減碼，等確認突破
時間接近收盤而未達目標：平倉
最重要的一件事：足跡圖不是讓你「找訊號」的工具，它是讓你理解「為什麼這個節點有效」的工具。先用 VP 和 AMT 找到有意義的位置，再用足跡圖確認微觀結構——兩者缺一不可。

系列結語

從市場微觀結構，到做市商的行為邏輯，到 AMT 的框架，再到 VP、TPO、VWAP，最後到足跡圖的實際交易——這七篇文章構成了一套完整的思維體系。

技術工具只是呈現資訊的方式，真正重要的是背後的邏輯：市場是一個持續的拍賣，被套的人必然逃跑，主力的行為在訂單流裡留下痕跡。看懂這些，你就比大多數人多了一個維度的資訊。
