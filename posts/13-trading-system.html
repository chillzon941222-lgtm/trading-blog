交易決策系統：從市場結構到進場確認的完整框架

學完所有工具之後，你需要的不是更多知識，而是一套清楚的觀察順序和決策邏輯。

chillzon2025

為什麼需要系統？

你已經學了 Volume Profile、AMT、足跡圖、CVD、VWAP——但在真實盤面前，腦中閃過太多指標，卻不知道先看哪個、怎麼整合，反而比什麼都不知道時更迷惘。

這篇文章的目的是：把所有工具排進一個有優先順序的決策流程，讓你每次面對市場都有清晰的問題要問，而不是憑感覺。

一句話概括整個系統的邏輯：

市場在拍賣。每一次碰到重要結構位置，你要問的只有一件事：
「市場接受這個價格（繼續往那個方向走），還是拒絕它（反轉回來）？」

CVD、OI、足跡圖，都是用來回答這個問題的證據。

觀察優先順序：從大到小，從結構到確認

不要同時看所有東西。按照以下順序建立你的情境，才不會被細節帶著走：

#觀察層次工具要回答的問題
1 結構位置 Volume Profile（VP）、VWAP 價格現在在哪？是在 Value Area 內、邊緣、還是之外？
2 拍賣情境 AMT：平衡 vs 不平衡 市場現在是在平衡區盤整，還是已經進入趨勢延伸？
3 拍賣測試 TPO、K 棒行為 市場在測試什麼？VAH、VAL、POC、Naked POC 還是前高/低？
4 足跡讀取 Footprint（Delta 差值） 在這個位置，是吸收（有人在扛）還是衰竭（沒人接）？
5 累積力道確認 CVD 主動買/賣力道是在持續累積，還是開始背離價格？
6 資金性質判斷 OI（未平倉量） 推動價格的是新倉位進場（真實方向），還是舊倉位平倉（假動能）？
7 進場決策 整合以上 接受（跟方向）還是拒絕（逆勢）？停損在哪？

核心場景一：VAH 測試

價格從 Value Area 內部上升，觸及 VAH（價值區高點）。這是最常出現的高品質設置之一。

VAH 是「大多數人認為公平的價格上緣」。當市場觸碰 VAH，就是在問：「買家願不願意在這麼高的地方繼續接受這個價格？」

訊號拒絕（Rejection → 做空）接受（Acceptance → 做多突破）
足跡 Delta 到達 VAH 後出現大量負 Delta（賣方主動砸），買方掛單不夠接 VAH 附近出現正 Delta 吸收（賣方在砸，但買方大量吃單，Delta 反而正）
CVD CVD 在價格創高時開始下彎（背離）——上漲缺乏買方支撐 CVD 持續上升，與價格同步——主動買力道持續累積
OI OI 下降：舊空單平倉推動的上漲（軋空），不是新多單進場，動能虛 OI 上升：新多單真實進場，方向性資金在建倉
行為 快速觸及 VAH 後長上影線，收盤拉回 VA 內 在 VAH 附近多次測試，每次拉回買盤積極，最終站上 VAH 並收盤於上方
決策 在 VAH 附近做空，止損設在 VAH 上方一個 Tick 突破 VAH 並確認收盤後做多，目標前期高點或下一個 HVN

核心場景二：VAL 測試

與 VAH 邏輯完全對稱。價格下跌觸及 VAL（價值區低點），市場在問：「賣家願不願意在這麼低的地方繼續賣？」

訊號拒絕（Rejection → 做多反彈）接受（Acceptance → 做空突破）
足跡 Delta 到達 VAL 後出現強烈正 Delta 吸收（賣方大量賣，買方全吃光） VAL 附近持續負 Delta，賣方主動且無買方承接
CVD CVD 背離：價格創新低但 CVD 不再下降（賣方力道耗盡） CVD 同步下降，主動賣力持續
OI OI 下降：舊多單止損（清倉），不是新空單，下跌動能快耗盡 OI 上升：新空單進場，真實做空資金在建倉
決策 在 VAL 附近做多，止損設在 VAL 下方 跌破 VAL 並確認後做空，目標前低或下一個 LVN

CVD × OI 矩陣：判斷動能的真實性

這是整個系統中最關鍵的過濾器。光看價格方向是不夠的，你需要知道推動價格的力量是「真的有人進場」還是「舊倉位在撤退」。

CVD 方向OI 變化解讀信號強度
CVD 上升（主動買增加） OI 上升 新多單進場，方向性資金真實建倉。多頭最強訊號 ★★★ 強多
CVD 上升 OI 下降 空單平倉（軋空）推動上漲，不是真實買盤。動能偏弱，小心反轉 ★★☆ 弱多
CVD 下降（主動賣增加） OI 上升 新空單進場，方向性資金真實做空。空頭最強訊號 ★★★ 強空
CVD 下降 OI 下降 多單止損（清倉）推動下跌，不是真實賣盤。下跌可能快速結束 ★★☆ 弱空
重要認知：軋空 vs 真實多頭

很多人看到價格大漲就追多，但如果 OI 同步下降，代表這波漲勢是空單被迫平倉，不是多方真實建倉。
軋空結束後，沒有新的多頭接手，價格往往快速回落。這就是為什麼 CVD 上升 + OI 下降 只是弱多訊號。

真正的趨勢發動，必然伴隨 CVD 方向 + OI 上升。

CVD 吸收 vs CVD 衰竭：足跡圖的細節讀取

CVD 告訴你累積的方向，但足跡圖讓你看到每一個價格節點發生了什麼。這兩個要一起讀。

吸收（Absorption）+ CVD 反向

價格下跌到某個區域，負 Delta 很大（主動賣很多），但 CVD 開始不再下降甚至微微上翹——代表有人在大量吸收（接走）賣壓，市場在這個位置建立了隱性支撐。

生活化比喻：想像一個拍賣現場，有人瘋狂出清貨物（主動賣），但每次東西一出來就有人快速買走（吸收）。表面上看起來很亂，但其實有人在偷偷積累。

這就是吸收：負 Delta + CVD 不繼續走低 = 底部正在形成

衰竭（Exhaustion）+ CVD 背離

價格持續創高，但 CVD 開始走平或下彎——代表主動買盤的力道在消退。即使價格還在漲，推力已經不夠了。

生活化比喻：一個人用力衝刺，跑得越來越慢，步伐開始虛，但還沒倒下。CVD 背離就是你看到他換氣越來越急、速度越來越慢——反轉訊號已經出現，只是市場還沒停下來。

這就是衰竭：正 Delta 越來越小 + CVD 上升斜率放緩/背離 = 頂部正在形成

整合應用：一個完整的交易流程範例

把所有觀察層次串在一起，假設 BTC 在做一個 VAH 測試：

步驟 ① 結構：BTC 現價 $68,500，VAH = $69,000，POC = $66,500，VAL = $64,000。
→ 價格在 Value Area 上緣，即將測試 VAH。

步驟 ② 拍賣情境：過去 3 天 BTC 在 $64,000–$69,000 盤整，這是平衡區。初次突破 VAH 需要謹慎確認。
→ AMT 情境：平衡中的邊緣測試，非趨勢市場。

步驟 ③ 測試行為：BTC 多次測試 $69,000，每次接觸後都有小幅拉回。
→ 市場在試探 VAH，尚未決定方向。

步驟 ④ 足跡：在 $69,000 附近，賣方砸出大量 Bid 成交，但 Ask 也有對應的大量吃單，Delta 接近中性偏正。
→ 吸收訊號出現，有人在 VAH 接走賣壓。

步驟 ⑤ CVD：CVD 隨每次 VAH 測試持續微升，沒有背離。
→ 主動買力在持續累積，非衰竭。

步驟 ⑥ OI：OI 在每次測試中緩步上升。
→ 強多訊號：新多單在建倉，非軋空。

步驟 ⑦ 決策：條件齊備（吸收 + CVD 同步 + OI 上升），等 K 棒收盤於 VAH 上方確認接受後做多。
止損：回到 VAH 下方（市場不接受，設置無效）。目標：前高或下一個 HVN。

常見錯誤與心態校準

錯誤原因正確做法
看到 CVD 上升就追多 忽略了 OI 下降（軋空），動能虛假 CVD 方向 + OI 必須同步確認
在 VA 內部交易 Value Area 內是雜訊區，雙向行為都有 只在 VA 邊緣（VAH / VAL）設置交易，不在中間找
看到大紅棒/大綠棒就追 可能是衰竭後的最後一棒，Gamma 風險最高 等 K 棒收盤確認，不追盤中動能
CVD 背離後立刻反手 背離是警告，不是入場訊號。市場可以繼續延伸 背離 = 警覺，等結構配合才進場（如回到 VAH 失守）
每個設置都做 缺少過濾，勝率自然下降 只做 CVD + OI 同步確認的設置，寧可少做，不做錯的

優先順序的最終版本

把上面所有內容壓縮成一張可以記住的清單：

進場前自問清單

1. 價格在哪？→ 結構位置（VA 邊緣優先，內部不做）
2. 大環境是平衡還是不平衡？→ AMT 情境
3. 市場在測試什麼關鍵位？→ VAH / VAL / POC / Naked POC
4. 足跡圖怎麼說？→ 吸收（逆向接貨）還是衰竭（推力耗盡）
5. CVD 同步還是背離？→ 背離 = 警告；同步 = 確認
6. OI 上升還是下降？→ 上升 = 新倉位（真實），下降 = 平倉（虛假）
7. 等 K 棒收盤確認，再進場。止損設在結構失效點。

如果以上任何一項不清楚，不做。

總結

交易系統的本質很簡單：在高機率的位置，用足夠的證據支持一個假設，然後用嚴格的止損驗證它。

  • 結構（Volume Profile）告訴你去哪裡等待
  • AMT 告訴你市場現在的狀態和預期行為
  • 足跡圖 + CVD 告訴你在那個位置發生了什麼
  • OI 告訴你這個動作是真的還是假的
  • 你的工作是等這四個層次對齊，然後只做有利風報比的設置

複雜不代表準確。你需要的不是更多指標，而是更少、但更確定的進場機會。
