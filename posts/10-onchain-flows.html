鏈上資金流動：錢往哪裡去，就往哪裡跟

Exchange Netflow、Stablecoin Supply Ratio、Miner Outflow——幣在流向哪裡？場內有多少火藥？礦工在賣了嗎？

chillzon2025

資金流動，比價格更早說話

上一篇我們看的是「持倉」——現在市場上的人賺還是賠。這篇我們往前一步：資金在移動的方向。

價格的漲跌，本質上是由資金的進出決定的。在加密市場，我們可以直接從鏈上看到這些資金在哪裡、往哪裡去。這是傳統市場做不到的事情。

這篇介紹三個最實用的資金流動指標：Exchange Netflow（交易所淨流量）、SSR（穩定幣供應比率）、Miner Outflow（礦工流出）。

Exchange Netflow：幣往交易所流，還是離開交易所？

交易所是買賣發生的地方。一般來說：

你想賣掉比特幣，你會先把幣轉進交易所，然後掛單賣出。
你想長期持有比特幣，你會把幣從交易所提出來，放到自己的冷錢包裡。

所以，「幣往交易所流入」和「幣從交易所流出」，分別代表完全不同的市場心態。

交易所流入增加（Positive Netflow）：持有者把幣送進交易所 → 準備賣出 → 賣壓增加

交易所流出增加（Negative Netflow）：持有者把幣從交易所提走 → 長期囤幣 → 流通供應減少 → 正面訊號
持續流入 = 市場在準備賣出｜持續流出 = 市場在囤幣，供應緊縮

實戰上怎麼用？

不要只看單日的數字，要看趨勢。幾天內持續大量流入，才有意義。單日一次性的大額流入有可能只是鯨魚在轉移錢包，不代表一定要賣。

搭配比對：大量流入但價格沒跌，代表有買盤在承接；大量流入加上價格下跌，代表賣壓正在主導市場。

SSR：場內有多少火藥等著進場？

SSR（Stablecoin Supply Ratio，穩定幣供應比率）衡量的是：相對於比特幣的市值，市場上有多少穩定幣的「火藥」等著進場？

計算方式：比特幣市值 ÷ 穩定幣總市值 = SSR

想像一下：如果市場上有很多人手上拿著 USDT、USDC，但還沒買入比特幣，這些錢就是「等著進場的火藥」。

SSR 低：相對比特幣市值，穩定幣很多 → 場內火藥充足 → 潛在買力強

SSR 高：相對比特幣市值，穩定幣很少 → 資金大多已入場 → 潛在買力薄弱
SSR 低 = 場外火藥多，市場有潛在買力；SSR 高 = 資金已入場，新進買力不足

SSR 是一個「未來買力」的衡量工具，不是即時訊號。它告訴你的是潛力，而非時機——場外火藥多，只代表有買入的條件，不代表明天就會漲。

Miner Outflow：礦工是沉默的長期賣壓

礦工每天挖出新的比特幣，這些幣就是市場的原生供應。礦工需要支付電費、機器折舊等成本，所以他們長期都是賣出方之一。

當礦工把幣轉移到交易所的速度加快，代表他們在賣出；當他們持有不動，代表他們對市場有信心，不急著賣。

Miner Outflow 增加：礦工把幣往交易所送 → 拋售壓力上升 → 短期利空

Miner Outflow 減少 / 礦工在囤幣：礦工對未來有信心，不急著賣 → 供應壓力降低 → 正面訊號

礦工投降（Miner Capitulation）：礦工大量虧損拋售，通常出現在熊市低點，是重要的底部訊號
礦工流出量在熊市底部達到高峰後消退，是歷史上重要的底部訊號之一

礦工投降是什麼？

當比特幣價格跌到礦工的挖礦成本以下，礦工開始虧損。為了支付電費，他們被迫把手中的幣全部賣掉，甚至關機停挖——這就是「礦工投降」。

這個過程非常痛苦，但它同時也在做一件事：把市場上最後一批被迫賣出的人清洗掉。歷史上，礦工投降後市場往往迎來較長期的底部確認。

Exchange Reserve：交易所的幣存量

Exchange Netflow 看的是「流量」（每天進出多少），Exchange Reserve（交易所總存量）看的是「存量」——目前各大交易所合計持有多少幣。

兩者相輔相成：Netflow 是短期變化的訊號，Reserve 是長期趨勢的結果。連續幾個月的 Netflow 流出，最終就會反映在 Reserve 持續下降上。

Reserve 持續下降：長期來看，持有者把幣從交易所撤出自己保管 → 流通供應減少 → 長期正面訊號

Reserve 突然急升：短期內大量幣湧入交易所 → 即將出現大規模賣壓 → 短期警示訊號

比較不同交易所的 Reserve 也很有意義：Coinbase 的流出增加通常和美國機構投資者的買入行為有關（因為很多 ETF 和機構用 Coinbase 保管），Binance 則代表全球散戶和亞洲資金的動向。

穩定幣的鑄造與銷毀

除了 SSR 之外，還有一個更細緻的觀察角度：穩定幣的鑄造（Minting）與銷毀（Burning）。

當大型投資者想進場買幣，通常會先把法幣換成穩定幣（USDT/USDC），這個換匯動作會導致穩定幣的總供應量增加——叫做「鑄造」。

當他們賣幣兌換回法幣，穩定幣被銷毀，總供應量減少。

穩定幣大量鑄造：有大量法幣正在換成穩定幣，準備進場 → 大量買力即將入場

穩定幣大量銷毀：資金在撤出加密市場，換回法幣 → 賣壓訊號

這個指標延遲性較高，但它反映的是「大錢在移動」——散戶不會鑄造幾億美元的穩定幣，能夠大規模鑄造的通常是機構或大型基金。

三個指標合起來看

這三個指標分別回答了三個不同的問題：

Exchange Netflow：持有者現在打算賣，還是囤？（行為方向）
SSR：場外還有多少資金準備進場？（潛在買力）
Miner Outflow：原生供應方在賣嗎？賣到什麼程度？（供應壓力）

搭配上一篇的持倉指標（SOPR、NUPL、HODL Waves），你就有了一個完整的鏈上分析框架：

底部訊號的典型組合：
SOPR 回升 + NUPL 在投降區 + HODL Waves 長期層增厚
+ Exchange Netflow 轉為流出 + SSR 開始下降 + 礦工投降後趨於穩定

頂部訊號的典型組合：
SOPR 高位轉跌 + NUPL 進狂喜區 + HODL Waves 短期層暴增
+ Exchange Netflow 持續大量流入 + SSR 偏高 + 礦工開始大量拋售

小結

Exchange Netflow：流入 = 準備賣，流出 = 囤幣，看趨勢而非單日
SSR：低 = 場外火藥多，買力潛在充足；高 = 資金已入場，動能可能減弱
Miner Outflow：礦工投降是熊市底部的典型訊號，投降後消退代表最後的拋壓出清

鏈上數據不能讓你每次都抄到最低點，但它能讓你在正確的週期位置做出更理性的決策——在別人恐慌的時候，你有數據支撐你的判斷。
